🛠️ Ręczna migracja użytkownika

Dla importu z innego bota: wpisz ID użytkownika Discord i dodaj ręcznie coinsy, XP lub levele.

📥 Import CSV (snapshot)

Format bez nagłówka: userId,level,xp,messages,voiceMinutes. Jeden błędny wiersz anuluje cały import.

Przykład wiersza: 123456789012345678,12,45,1000,250

🎖️ Role za level

Bot nadaje najwyższą rolę z progiem ≤ aktualny level i usuwa tylko inne role z tej mapy.

🏆 Leaderboard ekonomii

Przeglądaj ranking użytkowników po XP lub cebulionach z paginacją po 10 pozycji.
